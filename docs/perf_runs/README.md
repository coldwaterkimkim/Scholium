# Generated Performance Runs

`docs/perf_runs/` is intentionally reserved for local generated benchmark outputs.

Do not commit raw parser dumps, generated page images, or timestamped JSON run outputs from this directory. Keep reusable benchmark instructions and final decisions in:

- `docs/perf/PERFORMANCE_BASELINE_PLAN.md`
- `docs/perf/PARSER_BENCHMARK_SUITE.md`
- `docs/perf/PARSER_DECISION_20260507.md`

Historical March 2026 summary reports were moved to `docs/archive/perf_reports_20260329/`. They are evidence snapshots, not current source-of-truth architecture.
